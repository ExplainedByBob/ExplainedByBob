# ExplainedByBob

ExplainedByBob is an automated X (Twitter) bot designed to provide clear, concise explanations to user questions when explicitly prompted.

Users can invoke the bot by tagging the account and including the phrase:

**"Explain this Bob"**

---

## X (Twitter) Account

* **Handle:** @BobExplains_
* **Platform:** X (formerly Twitter)
* **Invocation Phrase:** `Explain this Bob`

### Example

```
@BobExplains_ Explain this Bob: What is inflation?
```

The bot responds with a direct, easy-to-understand explanation.

---

## Purpose

The purpose of ExplainedByBob is to:

* Provide straightforward explanations of questions or concepts
* Reduce complexity without oversimplifying
* Offer neutral, informational responses in public threads

The bot is designed to explain, not debate or persuade.

---

## Planned Functionality

* Monitor mentions of @BobExplains_
* Detect the trigger phrase "Explain this Bob"
* Extract context from the original post or thread
* Generate concise explanatory responses
* Apply rate limiting and abuse prevention
